# Dragon Warriors | Team #34046

Welcome to the official repository for **Dragon Warriors #34046**, an FTC (FIRST Tech Challenge) team from Ahmedabad International School.

This website serves as our team's digital portfolio, showcasing our robot design, outreach efforts, and team journey.

## Project Overview

The website is built as a **static site** using Vanilla JavaScript, HTML5, and CSS3. It utilizes [Vite](https://vitejs.dev/) as a lightweight development server and build tool.

### Features
- Responsive design for mobile and desktop.
- Interactive navigation and animations.
- SEO optimized with team-specific keywords.
- Fast-loading static assets.

## Project Structure

- `index.html`, `about.html`, etc.: Main page structures located in the root.
- `css/style.css`: Main stylesheet containing all global and component-specific styles.
- `js/main.js`: Main JavaScript file for interactivity (mobile menu, animations).
- `assets/`: Folder containing images, logos, and team photos.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Local Development
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser to the URL provided in the terminal (usually `http://localhost:5173/ftcdragonwarrior/`).

## Deployment

The site is configured for deployment to **GitHub Pages**. 

### Manual Deployment
1. Upload the following files/folders to the `main` branch:
   - All `.html` files
   - `css/`
   - `js/`
   - `assets/`
   - `.gitignore`
   - `package.json`
   - `vite.config.js`
2. In GitHub Settings > Pages, set the Source to "Deploy from a branch" and select `main` with the `/(root)` folder.
